**Объяснение ключевых моментов:**

1. **`Files.lines(Paths.get(filename))`** – читает все строки из файла
2. **`.parallel()`** – включает параллельную обработку стрима
3. **`filter(line -> line.contains(":"))`** – отфильтровывает людей без номеров (нет двоеточия)
4. **Преобразование имени**: приводим к виду с первой заглавной буквой и остальными строчными
5. **`Collectors.groupingByConcurrent()`** – потокобезопасный коллектор для группировки в параллельном режиме
6. **`Collectors.mapping()`** – преобразуем объекты Person в имена при группировке
